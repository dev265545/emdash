## What is Emdash?

Emdash is a desktop app for running AI coding agents in parallel. Each task runs in its own Git worktree, allowing exploration of multiple fixes or features at once.

## Key Features

- Run multiple AI agents in parallel
- Each task isolated in its own Git worktree
- Review diffs and merge what works
- Support for local projects and remote machines via SSH
- Compatible with various CLI agents: Claude Code, Codex, OpenCode, Gemini, Amp, and more
